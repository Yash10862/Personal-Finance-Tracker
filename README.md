#  Smart Expense Tracker (Java)

##  Description
A simple Java-based application that helps users track daily expenses, categorize spending, and manage budgets effectively.

##  Features
- Add and view expenses
- Delete expenses
- Category-wise summary
- Budget setting with alerts
- Data saved using file handling

## Technologies Used
- Java
- OOP Concepts
- File Handling
- Collections (ArrayList, HashMap)

##  How to Run

1. Compile:
   javac Main.java Expense.java ExpenseManager.java

2. Run:
   java Main

## Sample Features
- Budget warning when exceeded
- Expense summary by category

## 📂 Project Structure
src/ → Java source files  
data/ → Stores expense data
